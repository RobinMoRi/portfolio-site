# Portfolio site for Robin Moreno Rinding

This portfolio site is built using the following languages and tools:

### Frontend

- Nextjs
- Typescript
- Prismic
- Github/discord api integrations
- Shadcn and Tailwind
- Zustand

### Backend

- Python
- Fast API
- Pydantic

### Deployment and dev-environment

- Docker, Dockerhub
- Github workflows
- Hosted at Digital Ocean
